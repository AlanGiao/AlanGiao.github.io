CSS中值的类型与单位

发表于2022-07-22|更新于2022-07-22|webcss
|阅读量:

此为mdn web docs的笔记。

<length>, <color>, <image>, <position>, <function>等都是CSS中的数值类型(data type).

Note: CSS中，数值类型都是用尖括号括起来d。

长度(length)

绝对长度数值

cm, mm, Q(1/4 mm), in, pt(points), px(pixels)

相对长度数值

单位 含义
em 相对父元素的尺寸
ex 此元素 x 方向的尺寸
rem 相对根元素的尺寸
lh 当前元素的行高
rlh 相对于根元素的行高
vw 显示器宽度的1%
wh 显示器高度的1%

MDN文档中有更详细的叙述.

百分数(percentages)

当你给 width / font-size 的值设置成一个百分数的时候，它们将会把自己的值设置成其父元素的宽度/字号的相应百分比。

Note: 有的属性不接受百分数作为长度单位

颜色(color)

指定color有很多方式。

  • 关键字(color keywords)(这里是完整的颜色关键字列表).
  • 十六进制RGB数字指定，格式为#ffffff.
  • RGB 与 RGBA. 格式为 rgb(1,2,3)
  • HSL(Hue, Saturation, Lightness) 与 HSLA. 指定方法与RGB类似。

Note: RGBA 中的 A 代表alpha（透明度）通道。

Note: 这是一个色码转换器的链接, 支持 rgb、hsl、hsv转换。还有一个网站——中国色——有许多好看的国风色彩。
