除开上面的两种方法之外，background-size还可以取：

  • cover, 使得背景图片覆盖盒子的所有空白空间。可能会有部分图片超出盒子外；
  • contain, 使得背景图片恰好在盒子里面。可能会有留白；

设置背景图片的位置

语法：background-positoin: (x_cordinate, y_cordinate). 其中，左上角为(0,0).

此外，还可以将background-position的属性设置成

  • center
  • left
  • right
  • top
  • bottom

如，下面的CSS将会把背景图片放在左上角：

还可以用百分数来指定背景图片的位置： background-position: 50% 50% 就会把图片放在正中央。

还可以用图片离顶（底）端、左（右）端的距离来指定背景图片的位置（即所谓 4-value syntax）. 如background-position: top 10px left 20px;将会把背景图片放在离顶端10像素、左边20像素的位置。

设置背景图片是否随着页面滚动

这里需要使用 background-attachment属性。此属性可以有以下值：

  • scroll: 随着页面滚动；
  • fixed: 固定在页面上不动；
  • local: 将背景图片固定在其对应的元素上。在滚动页面的时候，图片会跟着元素一起走。

shorthand(简记表示法)

规则：

  • 先指定第一个背景的所有属性，然后用逗号分隔后，再指定第二个背景；
  • 必须在background-position后加一个/, 然后再指定background-size
  • background-color只能在最后一个逗号后指定；

边框(borders)

虽然可以分别设置边框的四个边，但是一般来说都是用shorthand同时设定四个框的样式。

If you want to specify one single edge of the border, you can use the longhand:

  • border-bottom-width
  • border-top-style
  • border-left-color
